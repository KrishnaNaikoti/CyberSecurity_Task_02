# Image Encryption Using Pixel Manipulation

## About The Task

This project is based on Image Encryption using Pixel Manipulation techniques.
The objective of this task is to understand how image encryption works by modifying pixel values using programming logic.

In this project, the image is encrypted by changing RGB pixel values using a secret key. The encrypted image can later be decrypted back to the original image using the same key.

The project is implemented using:
- Python

## What is Image Encryption?

Image encryption is the process of converting a normal image into an unreadable format to protect the image data from unauthorized access.

This project uses pixel manipulation for encryption. Since every image is made up of pixels containing RGB values, modifying those values changes the appearance of the image.

## Example

Original Pixel:
(100, 120, 140)

Encryption Key:
50

Encrypted Pixel:
(150, 170, 190)

The same key is used during decryption to restore the original image.
